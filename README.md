## 📚 Complete Study Programme

### German Language Curriculum (A1 → B2)

| Level | Module | Focus |
|-------|--------|-------|
| **A1.1** | Absolute Beginner I | Greetings, numbers, alphabet, basic sentences |
| **A1.2** | Absolute Beginner II | Family, colors, daily routine, simple questions |
| **A2.1** | Elementary I | Past tense, shopping, directions, appointments |
| **A2.2** | Elementary II | Hobbies, travel, health, phone conversations |
| **B1.1** | Intermediate I | Opinions, Konjunktiv II, news, work topics |
| **B1.2** | Intermediate II | Formal writing, complex grammar, TestDaF prep |
| **B2.1** | Upper Intermediate I | Academic German, DSH preparation, argumentation |
| **B2.2** | Upper Intermediate II | University-level German, advanced writing & speaking |

### Germany Guidance Programme

| Topic | Details |
